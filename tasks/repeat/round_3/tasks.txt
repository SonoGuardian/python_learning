1. Список квадратов
Создай список чисел от 1 до 10 и выведи новый список, где каждый элемент возведён в квадрат.

2. Минимум и максимум в списке
Пусть программа принимает от пользователя 5 чисел, сохраняет их в список и выводит на экран:
минимальное значение
максимальное значение

3. Переворот списка
Создай список слов и выведи его в обратном порядке (например, ["кот", "собака", "рыба"] → ["рыба", "собака", "кот"]).

4. Уникальные элементы
Пусть есть список чисел с повторами. Преобразуй его во множество, чтобы оставить только уникальные значения.

5. Проверка слова в списке
Создай список любимых фруктов.
Запроси у пользователя фрукт и выведи:

“Да, в списке” — если фрукт есть
“Нет, такого нет” — если отсутствует

6. Словарь с переводами
Создай словарь с переводом 5 английских слов на русский.
Пусть пользователь вводит английское слово, а программа выдаёт перевод (если он есть).

7. Подсчёт символов
Пусть пользователь вводит строку.
Создай словарь, где ключ — это символ, а значение — сколько раз он встречается в строке.

8. Слияние словарей
Есть два словаря, например:
a = {"x": 1, "y": 2}
b = {"y": 3, "z": 4}
Объедини их в один словарь.

9. Студенты и оценки
Создай словарь, где ключ — имя студента, а значение — список его оценок.
Напиши код, который считает средний балл каждого студента.

10. Телефонная книга
Сделай словарь, где ключи — имена, а значения — номера телефонов.
Добавь возможность:
вывести список всех контактов
найти телефон по имени
добавить новый контакт
